# GOABLE – Know Before You Go

GOABLE is a web-based platform that helps people with disabilities determine whether a public place is accessible before they travel. It provides personalized accessibility information and allows users to request assistance from registered volunteers when needed.

---

## Problem Statement

People with disabilities often face difficulties visiting public places because accessibility information is not easily available. This leads to uncertainty, dependence on others, and reduced independence.

---

## Our Solution

GOABLE enables users to:
- Select a public place.
- Specify their accessibility requirements.
- View a personalized accessibility score.
- Request assistance from nearby volunteers if required.

---

## Key Features

-  Personalized accessibility assessment
-  Public place accessibility information
-  Volunteer assistance system
-  User and Volunteer login
- Government/Admin portal to update accessibility data
-  Accessibility score based on available facilities

---

## Accessibility Parameters

- Wheelchair ramps
- Elevators/Lifts
- Accessible toilets
- Wide entrances
- Accessible parking
- Handrails
- Resting areas
